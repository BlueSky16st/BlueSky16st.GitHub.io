### tail 指令
``` Bash
// 显示文件最后 100 行
tail -n 100 XXX
// 显示文件最后 100 行同时实时读取文件
tail -n 100 -f XXX
```

### 查找文件中的内容
``` Bash
// 先搜索目录中匹配的文件, 然后再根据搜索结果查找包含了指定内容的文件
find ./ -name "*.csproj" | xargs grep -n "要查找的内容" | grep "要查找的内容"
// 这会搜索目录中所有文件中的内容, 效率较低
grep -nr "str" ./
```

### 查找指定端口的程序
``` Bash
netstat -tunlp | grep 端口
```

### 环境变量
``` Bash
// 修改
vim ~/.bash_profile
// 应用
source ~/.bash_profile
```

### 重启系统
``` Bash
shutdown -r +1 "这个linux要重启一下, 将在1分钟后重启"
```
